## ENV
#### PKG_REGISTRY
组件仓库地址,如: `http://172.178.1.204:2015/vue-components`

> 当前组件依赖远程组件时使用

#### PKG_INDEX
组件索引,默认为 `latest.json`

#### PKG_NAME
包名
> 打包/上传时使用

## run
``` bash
docker run \
    -e PKG_NAME xxx \
    -v $(pwd)/src:/app/src \
    -v $(pwd)/dist:/app/dist \
    vuep \
    1.2.3 [src/index.vue]
```
